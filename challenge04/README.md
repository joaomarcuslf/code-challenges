Escreva um algoritmo que retorne o número de valores únicos em `K`, e a quantidade de pares repetidos.

*Exemplo 01:*

```
Entrada: K = [ 1, 2, 4, 1, 3 ]
Saída: [ 3, 1 ]
```

*Exemplo 02:*

```
Entrada: K = [ 2, 40, 1, 3, -1, 3, 40, 5, 1 ]
Saída: [ 3, 3 ]
```
